def longest_increasing_run(nums):
    if len(nums) < 2:
        return 0

    longest = 0
    current = 1

    for i in range(1, len(nums)):
        if nums[i] > nums[i - 1]:
            current += 1
            if current > longest:
                longest = current
        else:
            current = 1

    return longest

Here is what it is doing:
- if the list has fewer than 2 numbers, it returns 0
- current keeps track of the increasing run we are currently in
- longest stores the biggest increasing run we have seen so far
- if the next number is bigger than the one before it, the run continues
- otherwise, the run breaks and current resets to 1
